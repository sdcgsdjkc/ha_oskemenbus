# Интеграция Oskemen Bus для Home Assistant

[![HACS Compatible](https://img.shields.io/badge/HACS-Compatible-brightgreen.svg)](https://hacs.xyz/)

Эта пользовательская интеграция предоставляет информацию о расписании автобусов в Усть-Каменогорске. Данные получаются через FastAPI-сервер, который запускается вместе с интеграцией.

## Возможности

- Получение расписания для указанной остановки.
- Фильтрация по номерам маршрутов (опционально).
- Запуск FastAPI-сервера для обработки запросов к API.
- Настройка через интерфейс Home Assistant (Config Flow).

## Установка

### Через HACS (рекомендуется)
1. Убедитесь, что у вас установлен [HACS](https://hacs.xyz/).
2. В Home Assistant перейдите в **HACS > Интеграции > Обзор и добавление репозиториев**.
3. Выберите опцию добавления репозитория по URL и укажите: https://github.com/sdcgsdjkc/ha_oskemenbus
4. Добавьте интеграцию и перезагрузите Home Assistant.
5. Перейдите в **Настройки > Устройства и службы** и добавьте интеграцию "Oskemen Bus". При настройке введите ID остановки и (при необходимости) номера автобусов через запятую.

### Ручная установка
1. Скачайте или клонируйте этот репозиторий.
2. Скопируйте папку `custom_components/oskemenbus` в директорию `config/custom_components/` вашего Home Assistant.
3. Перезагрузите Home Assistant.
4. Настройте интеграцию через интерфейс (Config Flow).

## Конфигурация

Интеграция использует Config Flow. При добавлении интеграции через интерфейс вам будет предложено:
- **stop_id** – ID остановки (например, `392193`)
- **buses** (опционально) – список номеров автобусов через запятую (например, `42`)

## Получение **Stop ID** для вашей остановки 🚏  

Чтобы узнать **Stop ID** нужной остановки, выполните следующие шаги:  

1. **Перейдите на сайт** [Oskemen Bus](https://oskemenbus.kz/maps).  
2. **Откройте инструменты разработчика**:  
   - Нажмите **F12** (или `Ctrl + Shift + I` в Chrome/Edge, `Cmd + Option + I` в macOS).  
3. **Выберите вашу остановку** на карте.  
4. **Переключитесь на вкладку "Сеть" (Network)**.  
5. В списке запросов найдите **GetScoreBoard** и кликните на него.  
6. Откройте вкладку **"Полезная нагрузка" (Payload)**.  
7. Найдите параметр:  
   ```json

Это и есть ваш Stop ID.

После этого можно использовать полученный Stop ID в настройках Home Assistant.

## Как это работает

1. **FastAPI-сервер**  
При настройке интеграции запускается FastAPI-сервер (через uvicorn) из файла `oskemenbus_parser/main.py`. Он доступен по адресу:  
`http://<IP_вашего_хоста>:8000/api/bus/schedule`  
Сервер обрабатывает запросы и возвращает расписание автобусов.

2. **Получение данных**  
Файл `bus_parser.py` отправляет POST-запрос к FastAPI-серверу с переданным ID остановки.  
Ответ сервера обрабатывается в файле `sensor.py`, где основной сенсор обновляется и отображает расписание.

## Зависимости
